### API Dependencies

**Endpoint:** `GET /api/metrics`
- **Query params:** `?agentId=@dev` (optional), `?limit=10` (default)
- **Response format:**
  ```json
  {
    "data": [
      {
        "id": "agent-1",
        "agent_id": "@dev",
        "team_id": "00000000-0000-0000-0000-000000000000",
        "status": "running",
        "latency_ms": 156,
        "success_rate": 99.3,
        "error_count": 1,
        "processed_count": 12450,
        "memory_usage_mb": 512,
        "cpu_percentage": 48.5,
        "recorded_at": "2026-03-13T23:45:00Z"
      }
    ],
    "total": 5,
    "limit": 10
  }
  ```

### Implementation Notes

1. **Use metricsService.subscribeToMetrics()**
   - Service already exists in `packages/dashboard/src/services/api/metrics.ts`
   - Handles polling every 5 seconds
   - Provides unsubscribe function for cleanup

2. **Hook Pattern**
   ```typescript
   // Create this hook
   export function useRealAgentMetrics(refetchInterval = 5000) {
     return useQuery({
       queryKey: ['agent-metrics-real'],
       queryFn: () => metricsService.getAgentMetrics(),
       refetchInterval,
       staleTime: 4000, // Refresh before next poll
     });
   }
   ```

3. **Component Integration**
   ```typescript
   // In AgentsMonitor.tsx
   const { data, isLoading, error } = useRealAgentMetrics();

   if (isLoading) return <AgentsSkeleton />;
   if (error) return <ErrorAlert message="Failed to load agents" />;

   return (
     <div>
       <h2>Agents <Badge variant="live">LIVE</Badge></h2>
       <p className="text-xs text-gray-500">
         Last updated {secondsAgo} seconds ago
       </p>
       {data?.map(agent => (
         <AgentMonitorCard agent={agent} key={agent.id} />
       ))}
     </div>
   );
   ```
