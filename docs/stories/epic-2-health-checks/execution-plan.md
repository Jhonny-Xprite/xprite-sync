## 7. Tech Stack & Requirements

### Test Framework Stack
- **Primary:** Jest (recommended for modern Node projects)
- **Fallback:** Mocha + Chai (if Jest incompatible)
- **Coverage:** Ensure minimum 80% statement coverage for framework code
- **Node Version:** 18+ LTS

### Health Check Scope
The health check suite must validate:
1. **Agent Definitions** - All 12 agents present, valid YAML syntax, required fields
2. **Framework Paths** - All critical `.aiox-core/` directories present
3. **Dependency Resolution** - All agent dependencies resolvable
4. **Configuration Integrity** - `.aiox-core/core-config.yaml` valid and accessible
5. **Script Execution** - Critical scripts (validate:agents, validate:structure) executable

### Pre-Commit Integration
- Hook type: **Bash script** in `.git/hooks/pre-commit`
- Fallback mechanism: Allow `--no-verify` override
- Performance requirement: Health checks complete in <30 seconds
- Logging: Output to `.git/logs/health-check.log`
