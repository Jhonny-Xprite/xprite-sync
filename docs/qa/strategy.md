# QA Strategy & Methodology

## 1. Principles
- **Shift-Left Testing:** Validation starts at the story definition phase.
- **Agentic QA:** Every task executed by an AIOX agent must pass a validation script.
- **Observability First:** Failures must be logged with enough context to allow automated recovery.

## 2. Test Layers

### Layer 1: Atomic (Scripts)
- Executed via `jest`.
- Focus on infrastructure script correctness.

### Layer 2: Molecular (Tasks)
- Validated via `aiox-validator.js`.
- Focus on task output compliance with schemas.

### Layer 3: Organism (Workflows)
- Multi-step validation.
- Focus on end-to-end flow integrity.

## 3. Quality Gates
| Gate | Runner | frequency |
|------|--------|-----------|
| Lint | `npm run lint` | Pre-commit |
| Typecheck | `npm run typecheck` | Pre-push |
| Structure | `npm run validate:structure` | Always |
| Agents | `npm run validate:agents` | Always |
